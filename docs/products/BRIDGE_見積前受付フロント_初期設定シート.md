# BRIDGE 見積前受付フロント 初期設定シート

## 入力方針

一人親方・小規模事業者でも迷わず設定できるように、自由入力を減らし、選択式・テンプレ選択式を中心にします。未確定の項目は「あとで設定」を選べるようにします。

## 1. 基本情報

| 項目 | 入力方式 | 選択肢・例 | 必須 |
| --- | --- | --- | --- |
| 事業者名 | 短文入力 | 株式会社BRIDGE 外構 | 必須 |
| 業種 | 選択式 | 外構 / リフォーム / 建築 / 塗装 / 設備 / 電気 / 水道 / その他 | 必須 |
| 対応エリア | 短文入力 | 例：○○市、△△町、車で60分圏内 | 必須 |
| 問い合わせ受付メール | メール入力 | info@example.com | 必須 |
| 公式LINE URL | URL入力 | https://lin.ee/xxxx | 任意 |

## 2. 受付内容

| 項目 | 入力方式 | 選択肢・例 | 必須 |
| --- | --- | --- | --- |
| 受付したい工事 | 複数選択 | 外構 / フェンス / カーポート / 庭 / 水回り / 内装 / 屋根 / 塗装 / その他 | 必須 |
| 対応できない工事 | テンプレ選択＋短文 | 高所作業 / 解体のみ / 遠方 / 小規模修理のみ / その他 | 必須 |
| 現地調査の有無 | 選択式 | 原則あり / 内容によりあり / 原則なし / あとで判断 | 必須 |
| 写真提出の希望 | 選択式 | できれば必要 / 必須 / 不要 / 内容により必要 | 推奨 |

## 3. 返信・営業時間

| 項目 | 入力方式 | 選択肢・例 | 必須 |
| --- | --- | --- | --- |
| 営業時間 | 選択式＋短文 | 平日9:00-18:00 / 平日・土曜9:00-18:00 / 不定休 / その他 | 必須 |
| 返信目安 | 選択式 | 24時間以内 / 2営業日以内 / 3営業日以内 / 繁忙期は遅れる場合あり | 必須 |
| 休日対応 | 選択式 | 対応しない / 確認のみ / 緊急時のみ / 内容により対応 | 任意 |

## 4. 予約金/着手金

| 項目 | 入力方式 | 選択肢・例 | 必須 |
| --- | --- | --- | --- |
| 案内有無 | 選択式 | 必要な場合のみ案内 / 原則案内する / 今は案内しない | 必須 |
| 名称 | 選択式 | 予約金 / 着手金 / 現地調査費 / 事前確認費 | 必須 |
| 金額 | 選択式＋金額入力 | 3,000円 / 5,000円 / 10,000円 / その他 | 必須 |
| Square決済リンク | URL入力 | 事業者が作成したSquareリンク | 任意 |
| 決済前の注意 | テンプレ選択 | 内容確認後に案内 / 日程確定後に案内 / 対象エリア確認後に案内 | 必須 |

## 5. キャンセル注意事項

| 項目 | 入力方式 | 選択肢・例 | 必須 |
| --- | --- | --- | --- |
| キャンセル注意事項 | テンプレ選択 | 前日までに連絡 / 当日キャンセルは再調整 / 現地到着後は費用が発生する場合あり / 個別に案内 | 必須 |
| 日程変更 | 選択式 | 可能 / 1回まで可能 / 内容により可能 / 個別に案内 | 任意 |
| 注意事項の表示 | 選択式 | 自動返信に入れる / 決済案内時に入れる / 両方に入れる | 推奨 |

## 6. 自動返信テンプレ選択

| 項目 | 入力方式 | 選択肢 | 必須 |
| --- | --- | --- | --- |
| 文面トーン | 選択式 | 丁寧 / 親しみやすい / 簡潔 | 必須 |
| 初回返信の目的 | 選択式 | 必要情報の確認 / 写真依頼 / 現地調査案内 / 日程確認 | 必須 |
| 予約金/着手金の書き方 | 選択式 | 必要時のみ控えめに案内 / 現地調査前に案内 / 日程確定後に案内 | 必須 |

## 7. 最小入力セット

最短で開始する場合は、以下だけ入力すればテストできます。

1. 事業者名
2. 業種
3. 対応エリア
4. 問い合わせ受付メール
5. Square決済リンクまたは「あとで設定」
6. 予約金/着手金の金額または「今は案内しない」
7. 現地調査の有無
8. 対応できない工事
9. 営業時間
10. 返信目安
11. キャンセル注意事項
