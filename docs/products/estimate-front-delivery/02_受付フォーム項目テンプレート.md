# 02_受付フォーム項目テンプレート

## 基本情報

1. お名前
2. メールアドレス
3. 電話番号
4. 連絡しやすい時間帯
5. 工事場所の市区町村

## 相談内容

1. 希望する工事内容
   - 外構
   - フェンス
   - カーポート
   - 庭
   - リフォーム
   - 塗装
   - 設備
   - その他
2. 現在の状況
   - 新規相談
   - 他社見積あり
   - 急ぎ
   - まだ検討段階
3. 希望時期
   - できるだけ早く
   - 1か月以内
   - 3か月以内
   - 未定
4. 予算感
   - 未定
   - 10万円未満
   - 10万〜50万円
   - 50万〜100万円
   - 100万円以上

## 写真・資料

1. 現場写真の提出可否
2. 図面・寸法メモの有無
3. 参考画像や希望イメージの有無

## 注意事項の確認

1. 対応エリア外の場合は受付できない場合があることを確認しましたか
2. 現地調査や事前確認に費用が発生する場合があることを確認しましたか
3. 日程変更・キャンセル時は早めに連絡することを確認しましたか

## 予約金/着手金を使う場合の確認

1. 決済前に対象工事・対応エリア・日程条件を確認すること
2. 決済後も工事可否や見積金額が確定するわけではないこと
3. 返金条件やキャンセル条件は事業者からの案内を確認すること
