# BRIDGE 見積前受付フロント 購入者向け納品パッケージ

このフォルダは、`BRIDGE 見積前受付フロント` を購入した事業者へ納品するための標準パッケージです。
既存の `BRIDGE ProofPack Starter` 納品物とは分離し、ProofPack Starter の自動納品文面・納品URL・運用を変更しません。

## 納品物一覧

| ファイル | 購入者ができること |
| --- | --- |
| `01_導入チェックリスト.md` | 初回設定、テスト、公開前確認の順番を確認できます。 |
| `02_受付フォーム項目テンプレート.md` | 見積前受付フォームに入れる質問項目をコピーできます。 |
| `03_自動返信テンプレート.md` | 顧客向け自動返信、事業者向け通知、予約金/着手金案内の文面をコピーできます。 |
| `04_運用ルール_免責.md` | 対応範囲、禁止表現、購入者側で確認すべき注意事項を確認できます。 |
| `Product_Master_見積前受付フロント_sample.csv` | 自動納品の商品判定を設定する場合のサンプル行を確認できます。 |

## 納品メールに入れる案内

自動納品メールでは、購入者に以下を案内します。

1. 納品URL
2. パッケージに含まれるファイル一覧
3. 初回設定の順番
4. テストモードでの確認を先に行うこと
5. Square決済リンクや本番送信は購入者・運用者が内容確認後に切り替えること
6. 本商品が売上保証、成約保証、法律・税務・契約判断を提供しないこと

## 本PRで行わないこと

- 本番購入
- 本番メール送信
- Square本番設定の作成・変更・削除
- Cloudflare環境変数の変更
- GAS Script Propertiesの変更
- 既存ProofPack Starterの納品URL、納品文面、商品判定の変更
